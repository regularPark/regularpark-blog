# 점심 뭐 먹지

### 1일차 회고

TDD와 타입스크립트 조합. 둘 중 어느 하나도 제대로 못하는 내게 엄청난 시련😭. 타입스크립트의 type과 interface의 차이를 학습했다.

type과 interface 대체 뭐가 다른데??

type alias는 어떤 형태로도 선언 가능하나, interface는 객체 형태로만 선언 가능하다. 또한 interface는 재선언하면서 확장이 가능하지만 type alias는 그렇지 않다.

```jsx
interface Person {
  name: string;
}

interface Person {
  age: number;
}

const user: Person = {
  name: 'IU',
  age: 30,
};
```

view와 연결을 해봐야하는데 공부가 필요할듯 싶다.

### 4일차 회고

localStorage를 구현하는 데 상당히 애를 먹었다. **계속 드라이버를 하고 싶어하는 패트릭:)**

열정적인 패트릭이 노트북을 가져가는 바람에 생각의 흐름을 따라가는 데 애를 좀 먹었다.

옆에서 보고 있다가 아이디어가 떠오르면 그걸 응용해서 구현하는 그런 형태가 계속 이어졌는데. 막판에 이르러서는 그것마저도 쉽지 않았다.

절대적으로 개발시간이 부족했던 것이 이번 미션 구현에 있어서 많은 걸림돌이 되었다. 삼일절에 쉬지 않았다면, 하는 생각이 계속 맴돌았다.🥲
